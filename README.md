# Basic Symfony contact form

Az applikáció maga egy egyszerű PHP-ban íródott contact form kitöltésére, és annak kezelésére képes.

### Szükséges programok
<ul>
    <li>php>=8.0</li>
    <li>composer>=2.6</li>
    <li>symfony>=5.8</li>
    <li>npm>=8.15</li>
</ul>

### Segédlet
```npm run``` <br>
```symfony server:start```
